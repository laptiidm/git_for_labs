**Класи (Widgets):**

* **`MyApp`**:
    * це кореневий віджет застосунку, який налаштовує основні параметри застосунку.
    * Використовує `MaterialApp` для налаштування теми, заголовка та головного екрана.
* **`MyHomePage`**:
    * це головний екран застосунку, який відображає лічильник та кнопку.
    * Є `StatefulWidget`, оскільки його стан (значення лічильника) може змінюватися.
* **`_MyHomePageState`**:
    * клас, який керує станом віджета `MyHomePage`.
    * Містить логіку для оновлення лічильника.
* **`MaterialApp`**:
    * віджет, який налаштовує основні параметри застосунку (тему, заголовок, головний екран).
* **`Scaffold`**:
    * віджет, який надає базову структуру екрана (панель заголовка, тіло, плаваюча кнопка).
* **`AppBar`**:
    * віджет, який відображає панель заголовка у верхній частині екрана.
* **`Center`**:
    * віджет, який центрує свій дочірній віджет.
* **`Column`**:
    * віджет, який розташовує свої дочірні віджети вертикально.
* **`Text`**:
    * віджет, який відображає текст.
* **`FloatingActionButton`**:
    * віджет, який відображає плаваючу кнопку.
* **`Icon`**:
    * віджет, для відображення іконок.
* **`ColorScheme`**:
    * віджет, для налаштування кольорової схеми.
* **`ThemeData`**:
    * віджет, для налаштування теми додатку.

**Методи:**

* **`main()`**:
    * точка входу застосунку.
    * Запускає застосунок за допомогою `runApp()`.
* **`build(BuildContext context)`**:
    * метод, який будує інтерфейс користувача.
    * Викликається при створенні або оновленні віджета.
* **`createState()`**:
    * метод, який створює об'єкт стану для `StatefulWidget`.
* **`_incrementCounter()`**:
    * функція, яка збільшує значення лічильника та викликає `setState()`, щоб оновити інтерфейс.
* **`setState()`**:
    * функція, яка оновлює стан віджета та перемальовує інтерфейс користувача.

**Властивості:**

* **`title`**:
    * заголовок застосунку або екрана.
* **`theme`**:
    * тема застосунку.
* **`home`**:
    * головний екран застосунку.
* **`appBar`**:
    * панель заголовка екрана.
* **`body`**:
    * тіло екрана.
* **`floatingActionButton`**:
    * плаваюча кнопка.
* **`children`**:
    * список дочірніх віджетів.
* **`mainAxisAlignment`**:
    * вирівнювання дочірніх віджетів на головній осі.
* **`colorScheme`**:
    * кольорова схема.
* **`seedColor`**:
    * початковий колір для кольорової схеми.
* **`inversePrimary`**:
    * колір, який інвертований від основного кольору.
* **`headlineMedium`**:
    * стиль тексту.
* **`onPressed`**:
    * функція, яка викликається при натисканні на кнопку.
* **`tooltip`**:
    * текст, який відображається при наведенні на кнопку.
* **`key`**:
    * ключ, який ідентифікує віджет.
* **`_counter`**:
    * приватна змінна, яка зберігає кількість натискань.

Цей список охоплює основні сутності, які використовуються в наданому коді.
